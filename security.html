Security & Trust

Enterprise-grade security from day one

CrowAgent is built on UK infrastructure with data protection, encryption, and access controls designed for regulated industry use.

AES-256 encryption

All data is encrypted at rest using AES-256 and in transit using TLS 1.3. Encryption keys are managed at the infrastructure level and rotated automatically.

UK data residency

All customer data is stored and processed in the United Kingdom. We do not transfer personal data outside the UK without explicit contractual safeguards. Our primary infrastructure is Supabase (UK region) and Vercel (EU/UK edge).

GDPR compliant

CrowAgent is registered with the Information Commissioner's Office (ICO). We maintain a full Data Processing Agreement (DPA). Rights supported: right to erasure, data portability, restriction of processing, and access. Contact hello@crowagent.ai for data requests.

Role-based access control

Platform access is controlled by organisation-scoped roles (Owner, Admin, Member). Row-Level Security (RLS) is enforced at the database level — users can only access data belonging to their organisation. MFA is available for all accounts.

ISO 27001 aligned

Our security controls programme is aligned with ISO 27001 principles. Formal certification is in progress. We conduct regular internal security reviews and apply OWASP best practices to all application development.

AI model data handling

AI inference (narrative generation, compliance analysis) uses Google Gemini via server-side API calls only. No customer data is sent to AI providers for training. AI inputs and outputs are not retained by model providers under our enterprise terms. All AI processing occurs on CrowAgent's infrastructure.

Vulnerability disclosure

If you discover a security vulnerability in CrowAgent, please report it responsibly to hello@crowagent.ai with the subject 'Security Disclosure'. We will acknowledge within 2 business days and aim to resolve confirmed vulnerabilities within 30 days.

Uptime commitment

CrowAgent targets 99.9% monthly uptime. For current system status, contact hello@crowagent.ai with subject 'Status Enquiry'. We will publish a status page at status.crowagent.ai in Q2 2026.
