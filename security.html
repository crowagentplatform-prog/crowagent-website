Data protection

How we protect your data

Encryption at rest and in transit

All data is encrypted using AES-256 at rest and TLS 1.3 in transit. No unencrypted data is stored or transmitted.

UK data residency

Your data is stored in UK-region infrastructure. We do not transfer personal data outside the UK/EEA without appropriate safeguards.

Role-based access control

Row-level security enforced at the database layer. Users only see data belonging to their organisation. Multi-factor authentication available.

Continuous monitoring

Real-time error monitoring and alerting via Sentry. Infrastructure monitored 24/7. Incident response procedures in place.
