MEES Compliance

What is MEES?
MEES (Minimum Energy Efficiency Standards) is UK legislation under SI 2015/962 that sets minimum EPC requirements for commercial properties. Currently, properties must reach EPC Band E to grant or renew a lease. A proposed Band C target for 2028 is under consultation but is not yet enacted law.
What is EPC Band C 2028?
Band C 2028 is a proposed regulatory target that would require all commercial properties to achieve EPC Band C by April 2028. This is subject to legislative confirmation and is not yet current law. CrowAgent Core helps you understand your position under both current Band E rules and the proposed 2028 target so you can plan ahead.
What properties are affected?
MEES applies to all privately rented commercial properties in England and Wales that are legally required to have an EPC. This includes offices, retail units, industrial premises, and warehouses. Properties rated F or G are currently non-compliant, while D and E rated properties would need improvement under the proposed 2028 Band C target.
How are MEES penalties calculated?
Penalties are calculated using the rateable value formula from SI 2015/962 regulation 39. For breaches under 3 months: 10% of rateable value (min £5,000, max £50,000). For breaches of 3 months or more: 20% of rateable value (min £10,000, max £150,000). CrowAgent never uses a flat £30,000 estimate — we always calculate from the actual rateable value.

PPN 002 / Social Value

What is PPN 002?
PPN 002 (Procurement Policy Note 002) is UK government policy requiring a minimum 10% social value weighting on all central government contract evaluations. It applies to all procurements above the relevant thresholds and requires suppliers to demonstrate social value commitments using the TOMs (Themes, Outcomes, and Measures) framework.
What is the 10% minimum?
Under PPN 002, at least 10% of the total evaluation score for central government contracts must be allocated to social value. Many local authorities and NHS trusts apply the same or higher thresholds. CrowMark helps you consistently hit this threshold with automated scoring and AI-generated narratives.
How is social value scored?
Social value is scored using the TOMs framework with proxy values from the Oxford Social Value Bank (2023-24 edition). Each measure has a monetary proxy value per unit. Your total social value is calculated as: SUM(proxy_value × quantity × duration_months). CrowMark automates this calculation using the latest Oxford SVB data stored in our database.

Technical

What data do you use?
EPC data comes from the MHCLG (Ministry of Housing, Communities and Local Government) Energy Performance of Buildings register — the official UK government source. Social value proxy values come from the Oxford Social Value Bank 2023-24 edition. Financial modelling uses HM Treasury Green Book 3.5% discount rate and Ofgem 2% energy escalation rate.
Is my data secure?
Yes. We use Supabase (PostgreSQL) with Row Level Security on all tables. All data is encrypted in transit (TLS) and at rest. We are protected by Cloudflare (DDoS, Bot Fight Mode, SSL Full Strict). Authentication uses Supabase Auth with optional TOTP MFA. We are ICO registered and GDPR compliant. See our Security page for full details.
How accurate is the EPC data?
We fetch directly from the MHCLG EPC register, which is the same data source used by energy assessors and local authorities. Our system caches data for 90 days and falls back to cached results if the register is temporarily unavailable. If no EPC is found, you can enter property data manually for analysis.
